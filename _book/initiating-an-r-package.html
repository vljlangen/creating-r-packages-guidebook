Creating R Packages: A Step-by-Step Guide

5 Initiating an R Package

In this chapter, we’ll learn how to turn an existing R project into an R package. We’ll use the usethis package to set up the necessary package structure and files.

This tutorial assumes that you have a freshly cloned, mostly blank GitHub repository, as explained in the previous chapter. We will now build upon it.

The following commands are to be run in the Console in RStudio.

5.1 Prerequisites

Before you start, make sure you have the devtools package installed, which includes usethis. If you haven’t installed it yet, you can do so by running:

install.packages("devtools")

5.2 Step-by-Step Guide

  1. Load the devtools Package

Load the devtools package, which automatically loads usethis, by running the following command in your R console:

library(devtools)
  1. Create the Package

Use the create_package function to initialize the package structure in your current working directory:

usethis::create_package(".")

This function will set up the basic structure needed for an R package. Your directory should now look something like this:

.
├── DESCRIPTION
├── NAMESPACE
├── R/
├── man/
├── .Rbuildignore
├── ...
